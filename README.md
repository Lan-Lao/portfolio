## Preview the portfolio locally
If the page is blank or styles/scripts seem missing, make sure you are serving it with a local web server (not opening `index.html` directly as a file).

From the project folder:

```bash
cd /workspace/portfolio
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000`

## Preview on GitHub Pages
Your production preview is:

- `https://lan-lao.github.io/portfolio/`

If you just pushed new changes and do not see updates yet:

1. Wait 1–5 minutes for Pages to rebuild.
2. Hard refresh the browser (`Ctrl+Shift+R` on Windows/Linux or `Cmd+Shift+R` on macOS).
3. Clear cache or open in an incognito window.
